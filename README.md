# 🌟 INTERNSHIP-TASKS-Todo-App

A full-stack to-do application built as part of an internship task — implemented with modern web technologies.

## 📦 What's Inside
- `todo-app/`: A production-ready Next.js + MongoDB + JWT authentication app
  - ✅ Frontend (App Router, Client/Server Components)
  - ✅ Backend (API Routes for auth & tasks)
  - ✅ TypeScript, Tailwind CSS, secure cookie-based auth

## 🚀 Get Started
1. Clone this repo:
   ```bash
   git clone https://github.com/Aishasiddiqui97/INTERNSHIP-TASKS-Todo-App.git
   ```
2. Go to the app folder:
   ```bash
   cd INTERNSHIP-TASKS-Todo-App/todo-app
   ```
3. Install & run:
   ```bash
   npm install && npm run dev
   ```
   → Open [http://localhost:3000](http://localhost:3000)

## 🔐 Environment Setup
Copy `.env.local.example` → rename to `.env.local`, then fill in:
- `MONGODB_URI`
- `JWT_SECRET`
